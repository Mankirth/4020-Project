Project Overview

This project blends web development, API integration, and data visualization to analyze ChatGPT's efficiency across multiple domains. We have created a multi-page interactive website that documents methodology, integrates Node.js middleware and WebSocket functionalities, and presents accuracy and performance insights using MongoDB and the ChatGPT API.

Introduction

ChatGPT has become a transformative tool in various fields, from education to professional work. However, understanding its efficiency across different knowledge domains‚Äîsuch as History, Social Science, and Computer Security‚Äîis crucial for informed adoption. This project measures ChatGPT's response accuracy, latency, and consistency across these domains to provide actionable insights.

Research Approach

  • Backend: Node.js with Express.js middleware
  • Database: MongoDB for storing evaluation results
  • API: ChatGPT API for prompt evaluation
  • Real-time Communication: WebSocket (ws) for live updates
  • Frontend: HTML5, CSS3, Vanilla JavaScript with client-side routing

Dataset Overview

Evaluation Domains:

  • History: Questions on historical events, figures, and timelines
  • Social Science: Questions on sociology, psychology, economics, and political science
  • Computer Security: Questions on cybersecurity concepts, threats, defenses, and best practices

Metrics Collected:

  • Response Time (latency)
  • Accuracy Score (manual evaluation)
  • Completeness of Response
  • Relevance to Query

Project: ChatGPT Efficiency Analysis

Objective

This project creates a comprehensive web-based platform to analyze ChatGPT's efficiency, accuracy, and performance across multiple domains (History, Social Science, Computer Security). The website integrates modern web technologies including Node.js middleware, WebSocket communication, and MongoDB database management.

Technical Architecture

Frontend

  • Client-Side Routing: Hash-based navigation (#/about, #/education, #/experience, #/project)
  • Dynamic Character: Animated assistant providing tips and project information
  • Responsive Design: Works seamlessly on desktop and mobile devices
  • WebSocket Integration: Real-time message display and status updates

Backend

  • Express.js Server: Handles HTTP requests and middleware processing
  • API Route: /api/add?a=2&b=3 validates inputs and returns computed results
  • WebSocket Server: Manages real-time bidirectional communication
  • MongoDB Integration: Persists evaluation data and chat history
